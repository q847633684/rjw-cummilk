# 领域概念（domain）概览

本目录存放**领域概念**类记忆：泌乳系统术语、符号、公式与权威文档引用。完整定义与公式以 `记忆库/docs/泌乳系统逻辑图.md` 为主，此处做可检索的术语表与索引。

## 核心术语与符号（来自泌乳系统逻辑图）

| 术语/符号 | 简短含义 | 权威引用 |
|-----------|----------|----------|
| **L** | 当前泌乳量（唯一状态量）；L ≤ 0 时泌乳结束 | [泌乳系统逻辑图](../docs/泌乳系统逻辑图.md) 第一、二、四节 |
| **E_tol(t)** | 统一耐受系数；实现中可带指数与开关 | 同上 第一节、第十二节 |
| **双池** | 左乳/右乳多对独立池；每对对应一个 RJW 乳房 hediff，共享 PairIndex | 第一节、第三节 |
| **吸收延迟** | 服药后由代谢率决定的延迟 tick，到点后才挂 Lactating 并进水 | 第一节 |
| **D(L, E)** | 每日衰减公式；B_T 由 baselineMilkDurationDays 反推 | 第一节、第十二节 |
| **进水周期** | 每 30 tick 双池由 L 进水；流速含 PressureFactor、R 倍率等 | 第三节 |
| **R（喷乳反射）** | 吸奶/挤奶后进水量 1.5～2.5 倍维持一段时间 | 第三节、第十二节 |
| **选侧规则** | 按对总满度从高到低，每对选较满一侧（相同时先左） | 第一节、DrainForConsume |

## 何时在本目录新增独立文件

- 某个术语需要单独展开（含公式、边界条件、与代码的对应）时。
- 新增概念（如新池类型、新公式）且已在 记忆库/docs 中定稿时。
