# 激素模型：催乳素维持与泌乳抑制

基于当前系统的「频繁排乳 → 泌乳期延长 / 长期不排乳 → 泌乳期自然结束」做可行性分析与设计。**结论：在现有 L/R/衰减/排乳刺激上扩展即可，无需废弃既有机制。** 公式与常数以 [泌乳系统逻辑图](../docs/泌乳系统逻辑图.md)、[四层模型升级-代码待办](../docs/四层模型升级-代码待办.md) 为准；本文规定「排乳反馈」与「泌乳抑制」的扩展方向与接口。

**⚠️ 与原有系统不兼容**：排乳反馈（lastDrainTick、抑制、Buffer、挤奶/吸奶触发维持）使泌乳期长度受「是否/何时排乳」影响，与原有「L 仅按时间衰减、排乳不参与 L 规则」的契约不一致。详见 [排乳反馈与原有系统不兼容说明](排乳反馈与原有系统不兼容说明.md)；关闭「长期不排乳抑制」与「排乳刺激缓冲」可恢复原行为。

## 一、目标逻辑

| 现象 | 激素解释 |
|------|----------|
| 频繁排乳 | 泌乳素维持 ↑ → 泌乳期延长 |
| 长期不排乳 | 泌乳抑制 ↑ → 泌乳期自然结束 |
| **催乳素** Prolactin | 决定泌乳维持 |
| **催产素** Oxytocin | 决定排乳效率 |
| **雌激素** Estrogen | 影响乳腺发育；刺激行为 → 催乳素 ↑ → 延长泌乳期 |

---

## 二、当前系统与激素对应

| 概念 | 当前实现 | 说明 |
|------|----------|------|
| 催乳素 / 泌乳维持 | **L**（currentLactationAmount） | L 驱动流速，L≤0 结束泌乳 |
| 催产素 / 排乳效率 | **R**（letdownReflexByKey） | 挤奶/吸奶时 R↑，进水流速×R、挤奶流速×R |
| 排乳对 L 的刺激 | AddMilkingLStimulus() | 每次排乳 L+= 微量，单次/每日上限；当前绑在 enableInflammationModel |
| L 衰减 | 每 200 tick：D(L,E)=1/(B_T·E)+k·L(+η·I) | 纯时间衰减，**尚未**与「是否近期排乳」挂钩 |
| 雌激素 | 无独立变量 | 容量/流速由 RJW Severity、基因、Conditions 等体现 |

**无需废弃**：L、R、D、双池、进水、耐受、炎症、喷乳反射、排乳刺激 均保留，只做**加法**。

## 三、缺口与扩展点

| 缺口 | 需新增 / 可加强 |
|------|-----------------|
| **长期不排乳 → 自然结束** | 记录 `lastDrainTick`；抑制项采用**连续函数**（软启动，见第六节），可选与乳池压力 P 联动（FIL）。 |
| **频繁排乳 → 维持 ↑** | 排乳刺激独立开关（不依赖炎症）；推荐**双通道**：即时 L+ + 维持缓冲减缓未来衰减（见第六节）。 |
| **雌激素 / 行为刺激** | 文档注明「乳腺发育含雌激素因素」；可选：RJW 事件调用 L 刺激接口。 |

## 四、落地步骤

### 阶段 1：排乳反馈与抑制（公式见第六节）

1. **上次排乳时间**：在 HediffComp_EqualMilkingLactating 或 CompEquallyMilkable 增加 `lastDrainTick`，DrainForConsume / OnGatheredLetdownByKeys 成功扣量后更新。
2. **长期未排乳抑制**：软启动，`SuppressionFactor ∝ days_since_drain²`，无阈值突变。
3. **排乳刺激**：从 enableInflammationModel 拆出为独立开关（如 enableProlactinStimulusFromMilking）；推荐双通道（即时 L+ + StimulusBuffer 减衰）。
4. **压力联动（可选）**：P 同时参与 L 衰减，形成「不排乳→满池→P↑→分泌变慢+L 衰减加快→结束」闭环。
5. 永久泌乳/动物常开**不应用**抑制。

### 阶段 2：UI 与文案

- L →「泌乳素水平」；R →「喷乳反射（催产素）」；新增「长期不排乳→泌乳抑制↑→自然结束」等说明。变量名可不改。

### 阶段 3（可选）

- 雌激素仅在容量/产奶效率说明中注明；RJW 行为事件调用 L 刺激接口。

## 五、需要废弃的吗？

**无。** 仅**新增**：lastDrainTick、SuppressionFactor（及可选 StimulusBuffer、P 参与 D_eff）；RemainingDays 仍由 L/D_eff 推导。

## 六、核心设计：软启动、双通道、压力联动

### 6.1 长期未排乳抑制（软启动）

- **问题**：「超过 N 天→衰减乘倍数」在阈值处突变，体验差。
- **做法**：抑制强度为未排乳时间的**连续函数**，例如  
  `SuppressionFactor = α × (days_since_drain)^β`，β≥1（如 β=2）。

| 未排乳时间 | 衰减影响（示例） |
|------------|------------------|
| 0.5 天 | 几乎无影响 |
| 1 天 | 轻微变快 |
| 2 天 | 明显变快 |
| 4 天 | 快速枯竭 |
| 7 天 | 基本结束 |

- **实现**：`days_since_drain = (now - lastDrainTick) / 60000f`；  
  `D_eff = D × (1 + SuppressionFactor)`，其中 SuppressionFactor 由上式计算。  
- **优点**：体验自然、符合回奶渐进、数值稳定。

### 6.2 排乳刺激（双通道）

| 类型 | 作用 | 生理类比 |
|------|------|----------|
| **即时刺激** | 直接 L+ | 催乳素瞬时升高 |
| **维持缓冲** | 减缓**未来**衰减 | 催乳素维持分泌、体质改善 |

- **即时**：保留 AddMilkingLStimulus 的 L+（perEvent / capPerDay 上限）。
- **缓冲**：新增 `StimulusBuffer`。每次排乳 Buffer += ΔB（有上限）；每 200 tick 衰减时 `D_eff = D × (1 - γ×Buffer)` 或 `D / (1 + γ×Buffer)`，Buffer 每游戏日衰减，排乳时再叠加。
- **效果**：频繁排乳 → 不仅延寿，且「更耐放置、不易突然回奶」（体质改善型）。

### 6.3 抑制与乳池压力 P 联动（FIL）

- **现状**：P 已抑制流速（PressureFactor）。
- **新增**：P **同时**参与 L 衰减。生理对应：FIL 随淤积升高。
- **实现**：P = 满度/撑大容量（已有）；  
  `D_eff = D × (1 + SuppressionFactor) × (1 + δ×P)`，高 P 时 L 衰减更快。
- **闭环**：不排乳 → 满池 → P↑ → 分泌变慢（现有）+ L 衰减加快（新增）→ 泌乳结束。

### 6.4 公式汇总（实现用）

统一符号：D = 基础日衰减，P = 满度/撑大容量，days = (now - lastDrainTick)/60000f。

- **抑制**：`SuppressionFactor = α × days^β`（α、β 为设置，β≥1）。
- **缓冲**（可选）：`D_eff = D × (1 + SuppressionFactor) × (1 + δ×P) / (1 + γ×StimulusBuffer)`，或等价形式；Buffer 排乳时 +ΔB、每游戏日衰减。
- 若不做压力联动，则去掉 `(1 + δ×P)`；若不做维持缓冲，则去掉 Buffer 项。

## 七、激素抽象层级（显示 ↔ 变量）

**原则**：医学名词保留在展示层，数值层抽象，避免系统复杂爆炸。

| 显示给玩家 / 文档 | 实际变量（代码层） |
|-------------------|---------------------|
| 催乳素水平 | L（currentLactationAmount） |
| 催产素强度 / 喷乳反射 | R（letdownReflexByKey） |
| 泌乳抑制 | SuppressionFactor（由未排乳时间²及可选 P 算出） |
| 乳腺发育 | CapacityFactor（容量/流速中的 Severity×系数、基因等，不单独变量） |
| 排乳刺激（即时） | AddMilkingLStimulus → L+ |
| 排乳刺激（维持缓冲） | StimulusBuffer（可选） |

## 八、其它影响泌乳的激素（生理参考）

除催乳素、催产素、雌激素外，下列激素在生理上影响泌乳。表中说明：**其生理效果**在本 mod 中是否已有**功能上的对应**（通过现有 L/R/P/健康/饥饿/药物等），以及是否建议**单独建模或使用医学命名**。

| 激素 | 生理作用 | 本 mod 中的功能对应 | 建议 |
|------|----------|---------------------|------|
| 孕激素 Progesterone | 孕期抑制泌乳；分娩撤退后启动泌乳 | 未显式 | 可作「分娩→L 初值」背景；一般不必单独变量 |
| 皮质醇/应激 | 长期应激抑制催产素、影响催乳素 | 未单独 | 可选：心情/应激 → R 或 L 衰减乘系数 |
| 多巴胺 | 抑制催乳素释放；拮抗剂可诱发泌乳 | **药物诱发泌乳**由 AddFromDrug 等体现（效果类似拮抗剂诱发） | 不必单独变量，也未用「多巴胺」命名 |
| 甲状腺 T3/T4 | 甲减降低产奶量 | **健康/SummaryHealth** 参与衰减与流速（效果类似甲减影响） | 视为健康一部分，未用「甲状腺」命名 |
| 生长激素 GH | 乳腺发育与产奶量 | **容量/流速**（RJW Severity、基因等）已含发育与产量影响 | 不必单独变量 |
| 胰岛素 | 代谢支持、乳糖合成 | **饥饿/BodyResourceGrowthSpeed** 影响流速与耗能 | 不必单独变量 |
| 血清素（乳腺局部） | 淤积时抑制合成 | **压力 P** 已抑制流速（PressureFactor），效果类似 FIL/局部抑制 | 不必单独变量 |
| 松弛素 | 孕期乳腺发育 | **Severity/基因/Conditions** 体现发育 | 不必单独变量 |

**小结**：  
- **不是**「每个激素都有专门变量、只差医学命名」——而是这些激素的**生理效果**（抑制/促进泌乳、影响流速等）已通过 **L、R、P、健康、饥饿、药物、容量** 等**现有机制间接体现**。  
- 可扩展且不增新变量：心情/应激 → 对 R 或 L 衰减乘系数；妊娠结束 → 触发 L 初值。  
- 若要在 UI/文档中标注「含甲状腺/代谢/应激等因素」，可在说明文案中注明，无需新增代码变量。
