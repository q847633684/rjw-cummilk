# 项目约定（conventions）概览

本目录存放**项目约定**类记忆：命名空间、Def 引用、翻译、Harmony、设置与存档等规范摘要。详细操作规范以 `.cursor/skills/rjw-cummilk-source` 为准，此处做可检索的摘要与索引。

## 与 skills 的关系

- **Skills**：规定「如何写」「如何检查」的步骤与清单，供 AI 执行。
- **本目录**：提炼为「是什么」的规则要点，便于快速查阅和标签聚合；可引用技能文件路径。

## 约定摘要（来自 rjw-cummilk-source）

| 领域 | 约定要点 |
|------|----------|
| **命名空间** | 与目录一一对应，file-scoped namespace；如 `MilkCum.Milk.Comps` ↔ `Source/MilkCum/Milk/Comps` |
| **Def 引用** | 本 mod Def 一律通过 **EMDefOf**；原版/它 mod 用 DefOf 或 `DefDatabase<T>.GetNamed(...)` |
| **翻译** | Keyed 用 `"KeyName".Translate()`，English + ChineseSimplified 同步；Def 文案可 Lang.SetDefaultLabel/Desc 或 DefInjected |
| **Harmony** | 挂接为主，不重写整段游戏逻辑；与《游戏已接管变量与机制清单》一致 |
| **设置与存档** | EqualMilkingSettings + Scribe，键用 `"EM.XXX"` 并保持稳定 |
| **与 Docs** | 重要设计在注释中引用 Docs 或规格；参数名与《参数联动表》一致 |

## 何时在本目录新增独立文件

- 某条约定需要展开为「为什么这样约定」或「反例说明」时。
- 与 rimworld-def-xml、其它技能有交叉时的摘要。
