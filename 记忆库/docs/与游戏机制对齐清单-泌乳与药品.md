# 与游戏机制对齐清单（泌乳与药品）

对照 RJW 壮阳药 / Humpshroom 的「完全利用原版逻辑」做法，记录 rjw-cummilk 中**已用上**与**未用上或未完全用上**的原版/游戏机制，便于后续对齐。

参考：RJW `Aphrodisiac.xml`、`Humpshroom.xml`（药品 + 效果 Hediff + 耐受 + 成瘾，均带 `SeverityPerDay`、`toleranceChemical`、`showDaysToRecover` 等）。

---

## 〇、游戏内药品信息面板字段 → Def 映射（完整）

游戏里点击药品时，信息面板会显示一整套成瘾/耐受/浓度等统计（例如原版大麻）。下表按**面板显示项**逐条对应到 Def/字段，并标出 rjw-cummilk 是否已配置。

| 面板显示项（示例名） | 数据来源 | rjw-cummilk 状态 |
|----------------------|----------|------------------|
| **药物浓度**（如 50%） | 效果 Hediff 或耐受 Hediff 的当前 severity（运行时） | ✓ Lactating 的 Severity 由原版 SeverityPerDay 驱动，L 与之同步 |
| **浓度降低速率**（如 100%/天） | 效果 Hediff 的 `HediffCompProperties_SeverityPerDay.severityPerDay` | ✓ 已通过 LactatingPatch 为 Lactating 添加 -0.10，一剂 0.5 约 5 天 |
| **有效浓度时间**（如 0.5 天） | 由 severity ÷ \|severityPerDay\| 计算 | ✓ 原版自动算；-0.10 时 0.5 即 5 天 |
| **每剂抗药性**（如 3%） | `IngestionOutcomeDoer_GiveHediff` 给耐受 Hediff 的 `severity` | ✓ 催乳素 0.044（按体型除）、露西拉辛 0.176 |
| **抗药性降低速率**（如 1.5%/天） | 耐受 Hediff 的 `HediffCompProperties_SeverityPerDay.severityPerDay` | ✓ EM_Prolactin_Tolerance：-0.015 |
| **成瘾恢复时间**（如 15.0 天） | 成瘾 Hediff：initialSeverity ÷ \|severityPerDay\|，且需 `showDaysToRecover` | ✓ EM_Prolactin_Addiction：0.5/0.015≈33 天，showDaysToRecover=true |
| **成瘾初始程度**（如 50%） | 成瘾 Hediff 的 `initialSeverity` | ✓ 0.5 |
| **成瘾需求降低速率**（如 50%/天） | NeedDef（DrugAddictionNeedBase）的 `fallPerDay` | ✓ Chemical_EM_Prolactin_Chemical：0.5 |
| **成瘾消费**（如 6.05 银/天） | 游戏按需求下降与药品价值计算，无单独 Def | 由原版根据 Need + 药品价值算出 |
| **成瘾需求间隔**（如 1.8 天） | needLevelOffset ÷ fallPerDay（天） | ✓ needLevelOffset=1、fallPerDay=0.5 → 2.0 天 |
| **化学物质**（如 大麻） | ChemicalDef 的 `label` | ✓ Prolactin（ChemicalDef label） |
| **成瘾性**（如 2%） | CompProperties_Drug 的 `addictiveness` | ✓ 催乳素 0.04（4%）；露西拉辛 1（100%） |
| **每剂满足需求**（如 90%） | CompProperties_Drug 的 `needLevelOffset`（0~1 即 0%~100%） | ✓ 1（100%） |
| **成瘾抗药性阈值**（如 15%） | CompProperties_Drug 的 `minToleranceToAddict` | ✓ 催乳素 0.03（3%）；露西拉辛 0 |
| **每剂成瘾度**（如 6%） | CompProperties_Drug 的 `existingAddictionSeverityOffset`（已有成瘾时每剂+严重度） | ✓ 催乳素 0.2；露西拉辛 1 |
| **随机过量几率**（如 0%） | CompProperties_Drug 的 `largeOverdoseChance` | ✓ 0.01（1%） |
| **成年安全服用间隔**（如 2.0 天） | CompProperties_Drug 的 `safeMarginDays`（原版社会性药品用，避免成瘾/过量的推荐间隔） | ✓ EM_Prolactin、EM_Lucilactin 均为 2 |

说明：上述所有「面板显示项」均来自原版药品/化学/成瘾体系；只要对应 Def 配好，游戏会自动显示。**当前已全部配置**（泌乳效果 Hediff 的 SeverityPerDay 通过 LactatingPatch 对原版 Lactating 追加，safeMarginDays 已加入两种催乳药品）。

## 一、已用上的原版/游戏机制

| 机制 | 本 mod 用法 | 说明 |
|------|-------------|------|
| **IngestionOutcomeDoer_GiveHediff** | 催乳剂/露西拉辛 `outcomeDoers` 给 Lactating、耐受 | 药品服用后给 Hediff，与原版一致 |
| **toleranceChemical** | GiveHediff 上写 `toleranceChemical>EM_Prolactin_Chemical` | 耐受影响本次给药 severity，原版逻辑 |
| **ChemicalDef + NeedDef** | EM_Prolactin_Chemical、Chemical_EM_Prolactin_Chemical | 成瘾需求、化学物质体系 |
| **CompProperties_Drug** | chemical、addictiveness、minToleranceToAddict 等 | 成瘾概率、需求偏移 |
| **耐受 Hediff** | EM_Prolactin_Tolerance：**SeverityPerDay** + **DrugEffectFactor** | 完全原版，衰减与药效系数 |
| **成瘾 Hediff** | EM_Prolactin_Addiction：**SeverityPerDay**、**showDaysToRecover** | 完全原版 |
| **短期 High** | EM_Prolactin_High：**SeverityPerDay -1** | 快速消退，原版 |
| **乳腺炎等** | EM_Mastitis：**SeverityPerDay** | 原版线性衰减 |
| **TryMergeWith** | 同 def 的 Hediff 合并时 Severity 叠加 | 原版合并逻辑；mod 在此之上补「L 进水」与补差 |
| **HediffComp_SeverityPerDay** | 原版 Lactating 通过 **LactatingPatch** 追加 `severityPerDay=-0.10`；CompPostTick 中 L 跟随 Parent.Severity | 底层原版驱动时间，上层 L 镜像；RemainingDays 有 SeverityPerDay 时用 Parent.Severity 与面板一致 |

## 二、未用上或未完全用上的机制（相对 RJW/原版）

### 1. ~~泌乳效果 Hediff 未配 SeverityPerDay~~（已解决，见第一节）

### 2. 药品中 hediffDef 引用名（已统一为 Lactating，见第一节）

### 3. 泌乳 Hediff 的 stages 未用 XML 驱动

| 项目 | RJW/原版做法 | 本 mod 现状 |
|------|--------------|-------------|
| **阶段与 capMods** | HediffDef 的 `<stages><li>` 里写 statOffsets/capMods，原版读 CurStage 用 | 设计原则 5：stages 在 C# 里 **GenStages() 动态生成**，Def 的 stages 仅保留单条供 CurStageIndex 不越界；capMods 在 GenStage 里按设置写 | 若希望完全用 XML 驱动阶段，需改为 Def 中写多阶段 + comps，并减少 C# 动态 GenStages |
| **成瘾阶段** | 成瘾 Hediff 的 stages 写满足/戒断、statOffsets | 本 mod 成瘾已有 stages，已用上 |

### 4. showDaysToRecover

| 项目 | 说明 |
|------|------|
| **成瘾** | 已用：EM_Prolactin_Addiction 有 `showDaysToRecover>true` |
| **效果 Hediff** | RJW 效果 Hediff 一般不显示「剩余恢复天数」；若希望泌乳也显示「剩余 X 天」，可在配好 SeverityPerDay 后考虑 showDaysToRecover（若原版支持对非成瘾 Hediff 显示） |

### 5. 其他可选原版机制

| 机制 | 说明 |
|------|------|
| **DrugDurationMultiplier（ statFactors ）** | 耐受阶段里已用，影响药效时长 |
| **divideByBodySize** | 给药 severity 按体型分摊，已用 |
| **IngestionOutcomeDoer_OffsetNeed** | Joy 等需求偏移，已用；带 toleranceChemical 已用 |

## 三、自定义实现汇总（与「纯原版」的差异）

| 功能 | 实现方式 | 若对齐原版可考虑 |
|------|----------|-------------------|
| 泌乳持续时间与剩余天数 | 有 SeverityPerDay 时由原版驱动，RemainingDays=Severity÷\|severityPerDay\|；否则 L÷(1/(B_T×E)) | 已对齐；双来源/永久泌乳仍用 L 公式 |
| 双池/产奶/流速/营养 | CompEquallyMilkable、池子、流速公式 | 保持自定义，与 SeverityPerDay 无关 |
| 吸收延迟 | WorldComponent_MilkCumAbsorptionDelay、EM_AbsorptionDelay、自定义 Tip | 保持自定义 |
| 泌乳阶段与 capMods | GenStages() 动态生成、EM_LactatingGain 单独 Hediff | 可选：部分 capMods 迁回 Lactating 的 Def stages（需与现有增益逻辑兼容） |
| 分娩/药物双来源 L | lactationAmountFromDrug / FromBirth、加权 B_T | 保持；若 Def 用 SeverityPerDay，可只驱动「药物」部分或统一一个等效 severityPerDay |
