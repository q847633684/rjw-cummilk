# 乳房容量模拟计算

本文档给出**乳房容量**（左/右池容量、总容量 maxFullness）的公式与示例计算，便于调参与排查。

---

## 1. 公式（代码对应）

- **前提**：`rjwBreastSizeEnabled == true`，且小人身上有 RJW 乳房 hediff 列表 `GetBreastList()`。
- **规则**：**每个 hediff 表示一对乳房**（单乳、多乳一致）。该 hediff 的容量 = `Clamp(Severity × coeff, 0, 10)`，同时计入左池与右池。
- **系数**：`coeff = rjwBreastCapacityCoefficient`（默认 2）。

| 量 | 公式 | 说明 |
|----|------|------|
| 左乳容量系数 leftFactor | 所有 hediff 容量之和 | 每个 hediff 的容量都计入左 |
| 右乳容量系数 rightFactor | 与 leftFactor 相同 | 每个 hediff 的容量都计入右 |
| 总容量 maxFullness | `leftFactor + rightFactor` = 2 × (所有 hediff 容量之和) | 左右池上限相同 |

- **泌乳期临时增大**：若开启「泌乳期增大 RJW 乳房」，泌乳时会对 RJW 乳房 hediff 的 `Severity` 临时 +0.15（上限 1），因此**同一小人泌乳期内的 Severity 更大，算出的容量也更大**；结束泌乳后恢复原 Severity，容量随之恢复。

---

## 2. 示例计算

设 `coeff = 2`（默认）。

### 双乳（2 个 hediff，每个表示一对），Severity 均为 0.5，coeff=2

- 每个 hediff 容量 = Clamp(0.5×2, 0, 10) = **1.0**
- totalCap = 1.0 + 1.0 = **2.0**
- leftFactor = rightFactor = **2.0**
- maxFullness = 2.0 + 2.0 = **4.0**

### 双乳（2 个 hediff），泌乳期临时 +0.15 后 Severity 均为 0.65，coeff=2

- 每个 hediff 容量 = Clamp(0.65×2, 0, 10) = **1.3**
- totalCap = 1.3 + 1.3 = **2.6**
- leftFactor = rightFactor = **2.6**
- maxFullness = 2.6 + 2.6 = **5.2**

### 双乳（2 个 hediff），容量系数 0.8，Severity 均为 0.5

- 每个 hediff 容量 = Clamp(0.5×0.8, 0, 10) = **0.4**
- totalCap = 0.4 + 0.4 = **0.8**
- leftFactor = rightFactor = **0.8**
- maxFullness = 0.8 + 0.8 = **1.6**

### 单乳（1 个 hediff 表示一对乳房），Severity=0.655，coeff=2

- 单乳容量 cap = Clamp(0.655×2, 0, 10) = **1.31**
- leftFactor = **1.31**，rightFactor = **1.31**
- maxFullness = 1.31 + 1.31 = **2.62**

---

## 3. 与其它逻辑的关系

- **池水位**：左池/右池当前水量上限分别为 leftFactor、rightFactor；总满度 = (leftFullness + rightFullness) / maxFullness。
- **灌满时间（估算）**：总流速/天 = 左池流速 + 右池流速（由 **defaultFlowMultiplierForHumanlike** 流速倍率参与，默认 2）。**单次剂量、Severity 0.5、容量系数 2、baseline=5** 时，总容量 2，约 **1 游戏日** 可灌满基础容量；**多剂**（L 更高、流速大）则更快，**更大乳房**（容量大、要填更多）则更慢。
- **挤奶工作量**：与 `MilkAmount` 和池满度有关，不直接读 leftFactor/rightFactor；见 `JobDriver_EquallyMilk.GetWorkTotal()`。
- **流速**：由 RJW 的 `fluidMultiplier`、**defaultFlowMultiplierForHumanlike**（流速倍率）、耐受等共同决定，与容量系数独立。
- **产乳逻辑（流速）**：与容量一致，**每个 hediff 表示一对乳房**，拆成两条池条目；每侧 `FlowMultiplier` 取该 hediff 的完整倍率（左右相同、不拆半）。总产乳/天 = 各虚拟侧进水之和（见 `UpdateMilkPools` / `CachedFlowPerDayByKey`）。

代码位置：`PawnMilkPoolExtensions.GetBreastCapacityFactors`（内部）、`GetBreastPoolEntries`；`CompEquallyMilkable.CompTick`（maxFullness 同步）、`UpdateMilkPools`（按侧进水）；`HediffWithComps_MilkCumLactating.GetFlowPerDay`（总流速显示）。
