## 阶段 1（安全升级）：压力软抑制 + 反射层 R

**目标**：引入 PressureFactor(P) 软停流；引入 R(t) 喷乳反射；**不改** L 衰减、**不引入** I。

### 1.1 设置与常数（EqualMilkingSettings / Defs）

- [x] 新增压力层参数（可设默认或 XML）：
  - `pressureFactorPc` = 0.9（P 超过此值流速开始明显下降）
  - `pressureFactorB` = 25～40（陡峭度）
  - 可选：`enablePressureFactor`（bool）便于关闭回退
- [x] 新增反射层参数：
  - `letdownReflexDecayLambda` = 0.02～0.05（每分钟衰减率 λ）
  - `letdownReflexStimulusDeltaR` = 0.3～0.6（挤奶/吸奶事件 ΔR）
  - 可选：`enableLetdownReflex`（bool）便于关闭回退

### 1.2 压力层 PressureFactor(P)

- [x] **公式**：PressureFactor(P) = 1 / (1 + exp(b·(P − Pc)))，P = Fullness / maxFullness。
- [x] **CompEquallyMilkable.UpdateMilkPools**：  
  当前 `if (Fullness >= maxFullness) flowPerTickScale = 0f` 改为  
  `flowPerTickScale *= PressureFactor(Fullness / maxFullness)`（若启用）。  
  保证 90% 前≈1，95% 后急剧降，100%≈0，不硬切。
- [x] **HediffWithComps_EqualMilkingLactating.GetFlowPerDay**：  
  当前 `if (Fullness >= maxF) return 0f` 改为  
  `return (baseFlowLeft+baseFlowRight) * PressureFactor(Fullness/maxF)`（若启用），与池子进水一致，饱食度/UI 一致。

### 1.3 反射层 R(t)

- [x] **状态存储**：在 **HediffComp_EqualMilkingLactating**（Lactating hediff 的 comp）中新增 `currentLetdownReflex` R ∈ [0,1]，默认可 1 或 0.5；Scribe 存档。
- [x] **衰减**：每 30 tick 与 UpdateMilkPools 同步执行  
  `R = R * exp(-λ * Δt)`，  
  Δt = 30 tick 折算为分钟（例如 30/60 = 0.5 分钟），λ 从设置读取。
- [x] **流速参与**：在 **UpdateMilkPools** 中计算 basePerDay/flowPerTickScale 后乘 R（若启用）；**GetFlowPerDay** 同理乘 R。
- [x] **事件触发**：挤奶/吸奶时（**Gathered**、**GreedyConsume** 调用的 **OnGathered**）  
  `R = min(1, R + ΔR)`，ΔR 从设置读取。

### 1.4 文档与审计

- [x] 逻辑图第六节「满池处理」补充：启用四层时采用 PressureFactor 软抑制，见第十二节。
- [x] 审计文档（泌乳系统逻辑图-代码审计）增加「升级阶段 1 实现」核对项。

## 阶段 2（中级）：炎症 I + L 衰减 + 挤奶真实化

**目标**：引入 I(t) 动态炎症与乳腺炎触发；L 衰减加入 η·I；挤奶时刺激 R、微幅 L、降 P（P 已由 Drain 自然降低）。

### 2.1 炎症状态 I(t)

- [x] **状态存储**：在 **HediffComp_EqualMilkingLactating** 或 **CompEquallyMilkable** 中新增 `currentInflammation` I ≥ 0；Scribe 存档。
- [x] **离散更新**（每 30 tick）：  
  `I_new = I + Δt·(α·P² + β·Injury + γ·BadHygiene − ρ·I)`，  
  P = Fullness/maxFullness；Injury/BadHygiene 由现有或新写辅助方法提供；α,β,γ,ρ 入设置；Clamp I ≥ 0。
- [x] **乳腺炎触发**：若 I > I_crit（如 1.0）则触发乳腺炎（挂 Hediff 或走现有乳腺炎逻辑）；可替代或与现有「满池 MTB」并存，由设计决定。
- [x] 设置：`inflammationAlpha`, `inflammationBeta`, `inflammationGamma`, `inflammationRho`, `inflammationCrit`。

### 2.2 L 衰减加入 η·I

- [x] **HediffComp_EqualMilkingLactating.GetDailyLactationDecay**：  
  原 `D(L,E) = 1/(B_T·E) + k·L` 改为  
  `D = 1/(B_T·E) + k·L + η·I`，  
  η 为设置项（炎症抑制因子）；永久泌乳/动物时可不加 η·I 或按现有逻辑。
- [x] 设置：`lactationDecayInflammationEta`。

### 2.3 挤奶真实化

- [x] **R**：阶段 1 已做挤奶时 R += ΔR。
- [x] **L 微幅刺激**：挤奶/吸奶后 `L += tinyStimulus`，带**上限**（例如单次不超过 0.05、每日不超过 0.2 等），防止无限循环；在 **OnGathered** 或 Gathered 调用的 Lactating 方法中写入 currentLactationAmount。
- [x] 设置：`milkingLStimulusPerEvent`, `milkingLStimulusCapPerDay`（或单次/每日 cap）。

### 2.4 文档与审计

- [x] 逻辑图第十二节 12.8、12.9、12.10 与实现对照；审计文档更新。

## 阶段 3（高级）：H(L)、组织适应、质量（可选）

**目标**：非线性有效驱动力 D_eff = L·H(L)；可选组织适应 F_max；可选质量系统。

### 3.1 激素层 H(L)

- [x] **公式**：H(L) = 1 − exp(−a·L)，a > 0（建议 0.5～1.5）；D_eff = L·H(L)。
- [x] 所有使用 L 参与流速的地方改为使用 D_eff（UpdateMilkPools、GetFlowPerDay 等）；L 仍为唯一长期状态，仅“驱动”改为 D_eff。
- [x] 设置：`hormoneSaturationA`，可选 `enableD_eff` 开关（代码中为 enableHormoneSaturation）。

### 3.2 组织适应（可选）

- [x] **dF_max/dt**：θ·max(P−0.85, 0) − ω·(1−P)；每 30 tick 或每日更新 maxFullness（或等效容量），Clamp 在合理范围；需与现有容量来源（RJW Severity×系数）协调，避免冲突。
- [x] 设置：`enableTissueAdaptation`, `adaptationTheta`, `adaptationOmega`, `adaptationCapMaxRatio`。

### 3.3 质量系统（可选）

- [x] **MilkQuality = f(Hunger, I)**：质量 = 饱食度 × (1 − 炎症权重×I)，Clamp 至 [0,1]；未启用时视为 1。
- [x] **乳腺炎**：启用质量时有效 I_crit = I_crit×(1 + milkQualityProtectionFactor×MilkQuality)，质量高则更难触发乳腺炎。
- [x] **UI**：启用时在泌乳 hediff 悬停中显示「乳汁质量：x%」。
- [x] 设置：`enableMilkQuality`, `milkQualityInflammationWeight`, `milkQualityProtectionFactor`。

- [x] 逻辑图第十二节 12.3、12.11 与实现对照；审计文档更新。
