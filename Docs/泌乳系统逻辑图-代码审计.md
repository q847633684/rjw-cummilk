## 十一、差异汇总与建议

### 1. 已修正 / 设计更新

- **挤奶/吸奶选对与选侧**  
  - **当前设计**：按「哪对最满」优先（对总满度从高到低），同对内先扣较满一侧，**左右相等时先左**（与性别/变性/无性别无关）。  
  - **代码**：`DrainForConsume` 中 `byPair = ... OrderByDescending(g => g.Sum(e => GetFullnessForKey(e.Key)))`，`preferLeft = true`。  
  - 文档与审计已同步更新。

### 2. 必须修正（与文档明确不一致）

（无）

### 3. 需产品/设计确认（文档与实现二选一）

（无。挤奶/吸奶已统一为：按对总满度优先、跨对取满当次需求量，文档第五节已更新。）
