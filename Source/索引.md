# rjw-cummilk Source 目录索引

按文件夹列出各 `.cs` 文件中的**类型**与**主要函数**及其功能简述，便于快速定位与检索。

## 目录层级与约定

**根目录**：`Source/`  
**主模块**：`MilkCum/`（Core、Fluids、Harmony、Integration、UI、Debug）

```
Source/
├── MilkCum/                    # 主程序集命名空间 MilkCum.*
│   ├── Core/                   # 入口、DefOf、建筑
│   │   ├── Settings/           # 全局设置（MilkCumSettings）
│   │   ├── Utils/              # EventHelper、Lang、VersionDiffHelper
│   │   └── Constants/          # Constants、PoolModelConstants
│   ├── Fluids/                 # 统一体液系统（Lactation + Cum + Shared）
│   │   ├── Lactation/          # 乳汁
│   │   │   ├── Comps/          # Comp、CompProperties（池、精液桶等）
│   │   │   ├── Hediffs/        # Hediff*（泌乳、吸收延迟、耐受、增益）
│   │   │   ├── Data/           # 数据与模型（MilkSettings、Breastfeed、Gene 等）
│   │   │   ├── Helpers/        # 扩展、工具
│   │   │   ├── Jobs/           # JobDriver
│   │   │   ├── Givers/         # WorkGiver / JobGiver
│   │   │   ├── Thoughts/       # ThoughtWorker
│   │   │   └── World/          # WorldComponent
│   │   ├── Shared/             # 两者共享
│   │   │   ├── FluidPool/      # LactationPoolState、BreastPoolEntry
│   │   │   ├── FlowModel/      # 流速模型占位
│   │   │   ├── PressureModel/  # 压力模型占位
│   │   │   └── NutritionModel/ # 营养模型占位
│   │   └── Cum/                # 精液（MilkCum.Fluids.Cum.*）
│   │       ├── Comps/          # 精液相关 Comp
│   │       ├── Hediffs/        # 精液 Hediff
│   │       ├── Cumflation/     # Cumflation 子模块
│   │       └── Bukkake/        # Bukkake 子模块 等
│   ├── Harmony/                # Harmony 补丁
│   ├── Integration/            # 与其它 mod 的集成（RJW、Vanilla、PipeSystem、DubsBadHygiene）
│   ├── UI/                     # 设置界面、表格、列、窗口
│   └── Debug/                  # 调试工具（DevTools 等）
```

**约定**：索引章节顺序与上述层级一致；新增 `.cs` 请放入对应目录并在下表中补充一行。
